System overview

The edus2TM system combines a simple RFID reader, some RFID tags, and a laptop computer to simulate bedside ultrasound. The software is available free of charge. Currently some set up is required, contact Paul Kulyk for assistance. As questions arise they will be added to the wiki. A turn-key solution is being worked on that will allow the user simply to boot the laptop and begin scanning.

Equipment needed

  • A 125 kHz USB-based RFID scanner. To work with the software provided it must output a string for each tag read followed by a newline. We used an Olimex RFID-MOD125
  • Some 125 kHz RFID tags. We purchased both the wallet-sized cards as well as the small discs. We found the wallet cards worked well under the mannequin skin, and the discs were useful for some obstetrical scans when attached to the mannequin with skin-coloured tape.
  • A laptop running Ubuntu Linux.
  • A 2 m USB cable to connect your computer to the probe.
  • (Optional) constructing something to house the RFID scanner in.

Usage

The steps are outlined here for general usage of the edus2TM system. As questions arise they will be answered in the wiki. Skip to the bottom for a video overview.

Step 0: Set up the scans

For each scan a card must be set up to trigger it. Each video will be associated with one unique RFID tag. Details on this can be found on the wiki. NOTE: This only has to be performed once.

Step 1: Set up the simulation

Insert the RFID tags corresponding to the scans needed for your simulation under the skin of the mannequin. Alternately, if the mannequin skin does not work smaller tags can be taped to the skin of the mannequin.

Step 2: Student scans as indicated

If a student feels an EDUS scan would aid his or her diagnosis, they simply pull the machine to the bedside and place the probe over the appropriate landmark.

For the image to be displayed they must know the appropriate landmark for the scan.

Step 3: Scan is displayed

Once the landmark is correctly located, the scan is display on screen for the student to interpret.
