System overview

The edus2TM system combines a simple RFID reader, some RFID tags, and a laptop computer to simulate bedside ultrasound. The software is available free of charge. Currently some set up is required, contact Paul Kulyk for assistance. As questions arise they will be added to the wiki. A turn-key solution is being worked on that will allow the user simply to boot the laptop and begin scanning.

Equipment needed

  • A 125 kHz USB-based RFID scanner. To work with the software provided it must output a string for each tag read followed by a newline. We used an Olimex RFID-MOD125
  • Some 125 kHz RFID tags. We purchased both the wallet-sized cards as well as the small discs. We found the wallet cards worked well under the mannequin skin, and the discs were useful for some obstetrical scans when attached to the mannequin with skin-coloured tape.
  • A laptop running Ubuntu Linux.
  • A 2 m USB cable to connect your computer to the probe.
  • (Optional) constructing something to house the RFID scanner in.
